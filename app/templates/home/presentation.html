Simple System of ODE's

\[ dSdt = \beta G - \alpha A \\ dAdt = \gamma S \\ dGdt = -\delta A \]

With simple ODE's comes great Dynamics

Nice and all, but what about data?

  • Nested optimization algorithm
    • One for finding the optimal ODE
      • Find optimal parameter set for fitting of the data
    • One for finding the optimal parameter values
      • Find optimal parameter values for fitting of the data
